eOffer/eMod Help Center

August 2013 Release Preview

6/14/2013

In the Augst 2013, GSA will be making some big changes to the eOffer/eMod System. Some highlights of this release are listed below, but we strongly advise readers to examine the entirety of the document to identify issues relevant to their business needs.

  • FSS Online is now fully compatible with Windows 7. Previously, any process related to saving a PDF would fail due to Windows 7 security settings. The system and user laptops have been updated enabling this functionality.

  • When a vendor submits a new offer against a GSA schedule, they will now be required to list a single NAICS code that represents the “Preponderance of Work” for their company. Additionally, some information that is requested from the offeror will is now automatically incorporated into the offer forms, if it is available from existing sources.

  • Vendors will now be able to fully process a termination or cancellation mod when their company had an expired SAM registration.

  • COs and CSs will now receive a copy all contract-related notifications that are generated by FSS Online.

  • Request for Clarification letters that are sent to a vendor regarding a contract modification will now include the correct contact information when that mod has been delegated to an ACO.
