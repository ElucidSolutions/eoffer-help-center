February 2014 Release Notes Updates

2/27/2014

In February 2014, GSA implemented changes to the eOffer/eMod System. You can read a detailed description of each update on the Release Notes website.

  • eOffer/eMod is now compatible with Firefox to support a wider ranger of users.

  • The Add Labor Category and/or Service Offering now requires Vendors to go through the Subcontracting Plan utilitiy to add a Subcontracting Plan to their to their Contract. Previously, users simply uploaded a handmade Subcontracting Plan.

    To learn more, read the Subcontracting Plan section under the eMod Resources category of the User Guide.

  • Vendors can now choose two full years for their Subcontracting Plan on Commercial Plans, whereas previously the utility only allowed Vendors to choose a one year term.

    To learn more, read the Subcontracting Plan section under the eMod Resources category of the User Guide.

The next release is in May of 2014. Check back to find out the latest updates.
