September 2014 Release Notes Updates

10/27/2014

In Sepetmber 2014, GSA implemented changes to the eOffer/eMod System. The follow changes have been made:

  • When a Vendor submits a new Offer under the CORP Schedule, eOffer will now ask if the vendor is migrating existing contract(s) to a consolidated contract under the Standard Responses section.

  • On the eOffer/eMod Homepage, there is now a list in the About eOffer section titled View Solicitations that contains links to MAS Schedule Solicitations, (previously found in the Learn More About: section), and Non-MAS Schedule Solicitations.

    MAS and Non-MAS Schedules can now be found on the eOffer/eMod homepage
